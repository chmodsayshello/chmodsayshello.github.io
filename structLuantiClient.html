Detailed Description

The LuantiClient struct represents a Luanti client in memory.

After allowcation, it should be overwritten with 0 (i.e. using bzero() or memset()) so all function pointers are NULL.
Set LuantiClient::username , LuantiClient::username_len and LuantiClient::callbacks accordingly before using LuantiClient_connect .
